# Assets

この ディレクトリには、アプリのアイコンやスプラッシュ画像を配置します。

## 必要なファイル:

- `icon.png` - アプリアイコン (1024x1024 px)
- `splash.png` - スプラッシュスクリーン (1284x2778 px)
- `adaptive-icon.png` - Androidアダプティブアイコン (1024x1024 px)
- `favicon.png` - Webファビコン (48x48 px)

現在はプレースホルダーとして空のディレクトリです。
Manus AIでビルドする際に、デフォルトのExpoアイコンが使用されます。
