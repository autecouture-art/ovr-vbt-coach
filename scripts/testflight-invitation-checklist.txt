# OVR VBT Coach - TestFlight テスター招待チェックリスト

## 前提条件

- [ ] App Store Connect でアプリが登録済み
- [ ] Manus ビルドが完了
- [ ] App Store Connect でビルドが表示される
- [ ] ビルドステータス: Ready to Test
- [ ] テスターのメールアドレス収集済み

## ステップ 1: Internal Testers グループ作成

- [ ] App Store Connect にログイン
- [ ] OVR VBT Coach を選択
- [ ] TestFlight タブをクリック
- [ ] Internal Testers セクションで + をクリック
- [ ] Group Name: Core Team を入力
- [ ] Create をクリック

## ステップ 2: テスター追加

- [ ] Internal Testers → Core Team をクリック
- [ ] Add Testers をクリック
- [ ] テスターのメールアドレスを入力
- [ ] Add をクリック
- [ ] テスター追加完了を確認

## ステップ 3: ビルドをテスターグループに追加

- [ ] TestFlight → Builds をクリック
- [ ] 最新のビルド（2.3.0 Build 1）をクリック
- [ ] Add Build をクリック
- [ ] Internal Testers → Core Team を選択
- [ ] Save をクリック

## ステップ 4: テスター招待メール確認

- [ ] テスターが招待メールを受け取ったか確認
- [ ] メール件名: You're invited to test OVR VBT Coach
- [ ] メール内のリンクが有効か確認

## ステップ 5: テスター指示送信

- [ ] テスターに指示書を送信
- [ ] TestFlight アプリダウンロード手順
- [ ] OVR VBT Coach インストール手順
- [ ] テスト項目（アプリ起動、BLE 接続、セッション記録、PR 検知）
- [ ] バグ報告方法

## ステップ 6: テスト監視

- [ ] App Store Connect で Testers セクションを確認
- [ ] テスターのステータスを監視
  - [ ] Invited（招待メール送信済み）
  - [ ] Installed（アプリをインストール）
  - [ ] Launched（アプリを起動）
  - [ ] Active（定期的に使用中）

## ステップ 7: フィードバック確認

- [ ] TestFlight → Feedback をクリック
- [ ] テスターからのフィードバックを確認
- [ ] バグレポートを分類
- [ ] 優先度を決定

## トラブルシューティング

### テスター招待が送信されない

- [ ] メールアドレスが正しいか確認
- [ ] テスターが Apple ID を持っているか確認
- [ ] App Store Connect で Resend Invitation をクリック
- [ ] スパムフォルダを確認

### テスターがアプリをインストールできない

- [ ] テスターが招待メールを受け取ったか確認
- [ ] TestFlight アプリがインストールされているか確認
- [ ] iOS バージョンが 15.1 以上か確認
- [ ] App Store Connect でテスターが追加されているか確認

### アプリがクラッシュする

- [ ] Crash Report を確認
- [ ] ローカルでクラッシュを再現
- [ ] バグを修正
- [ ] 新しいビルドを提出
