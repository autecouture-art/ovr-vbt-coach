# OVR VBT Coach - モバイルアプリ設計ドキュメント

## 概要

OVR VBT Coach は、Velocity-Based Training (VBT) センサーと連携し、リアルタイムで速度・パワーを計測するiOS向けモバイルアプリです。AIが1RM推定、PR検知、トレーニング調整を自動で行う完全自動VBTコーチアプリです。

**対象プラットフォーム**: iOS (iPad対応)  
**向き**: ポートレート (9:16) / ランドスケープ対応  
**デバイス**: iPhone / iPad  
**使用スタイル**: 片手操作を基本に、両手操作にも対応

---

## スクリーン構成

### 1. ホーム画面 (Home)
**目的**: トレーニングセッションの開始、今日の概要表示

**主要コンテンツ**:
- セッション開始ボタン（大型、視認性重視）
- 本日の累計ボリューム表示
- 直近のPR表示
- 今日の調子スコア（Readiness Assessment）
- クイックアクション（設定、過去セッション）

**機能**:
- セッション開始時に体重・調子を入力するダイアログ
- 前回のセッション情報を表示
- 推奨トレーニング内容の提示

---

### 2. モニター画面 (Monitor)
**目的**: VBT計測中のリアルタイム速度表示

**主要コンテンツ**:
- 巨大な速度表示（Mean Velocity / Peak Velocity）
- Velocity Loss バー（%表示）
- 現在のセット / レップ情報
- ROM、Rep Duration の小表示
- 推奨重量・レップ数の表示

**機能**:
- BLE接続状態表示
- セット終了時の自動PR検知
- ドロップセット時の次重量自動提案
- AMRAP モード対応（V-Loss アラート無効）
- 音声フィードバック

---

### 3. マニュアル入力画面 (Manual Entry)
**目的**: VBTセンサーなしの日のトレーニング記録

**主要コンテンツ**:
- 種目選択（ピッカー）
- 重量入力フィールド
- レップ数入力フィールド
- RPE スライダー
- テンポ入力（例: 2-0-1-0）
- セット種別選択（Normal / AMRAP / Drop / Superset）
- メモ入力フィールド

**機能**:
- 簡易e1RM推定表示
- セット記録後の自動PR検知
- VBTデータとの混在管理

---

### 4. LVP グラフ画面 (LVP Graph)
**目的**: 種目別の負荷速度プロファイル表示

**主要コンテンツ**:
- 種目選択タブ
- 散布図（負荷 vs 速度）
- LVP直線（回帰線）
- 速度ゾーン表示（Power / Strength Speed / Hypertrophy / Strength）
- 統計情報（Vmax, V1RM, R²）

**機能**:
- 過去データから自動計算
- 速度帯別の推奨負荷表示
- トレンド表示（改善/低下）

---

### 5. 履歴・カレンダー画面 (History)
**目的**: 過去セッションの閲覧

**主要コンテンツ**:
- カレンダービュー（セッション実施日は緑色）
- 日付選択時のセッション詳細表示
- セッション内のセット一覧
- セット内のレップ詳細

**機能**:
- スワイプで月移動
- セッション詳細の展開
- データのエクスポート（将来）

---

### 6. 週間サマリー画面 (Weekly Summary)
**目的**: 週間のトレーニング統計

**主要コンテンツ**:
- 週間ボリュームグラフ
- 先週との比較
- 種目別ボリューム
- 直近のPR一覧
- 平均速度トレンド

**機能**:
- 週単位での切り替え
- AIアドバイス表示

---

### 7. 設定画面 (Settings)
**目的**: アプリ設定の管理

**主要コンテンツ**:
- Velocity Loss 閾値設定
- 音声フィードバック ON/OFF
- 音声コマンド ON/OFF
- 動画記録 ON/OFF
- トレーニングフェーズ選択（Power / Hypertrophy / Strength / Peaking）
- 単位設定（メートル法 / ヤード・ポンド法）
- BLE デバイス管理
- データ管理（エクスポート / 削除）

**機能**:
- 設定の永続化（AsyncStorage）
- BLE デバイスペアリング管理

## 主要ユーザーフロー

### フロー 1: VBT セッション実施
1. ホーム画面で「セッション開始」をタップ
2. 体重・調子を入力するダイアログが表示
3. 種目を選択
4. BLE デバイスに接続
5. モニター画面で速度をリアルタイム表示
6. セット終了時に自動PR検知
7. 次セット開始 or セッション終了

### フロー 2: マニュアル記録（ノンVBT日）
1. ホーム画面で「マニュアル入力」をタップ
2. 種目、重量、レップ数、RPE を入力
3. 「セット記録」をタップ
4. 自動PR検知
5. 次セット入力 or セッション終了

### フロー 3: LVP確認
1. LVP グラフ画面を開く
2. 種目タブを選択
3. グラフ上で負荷速度プロファイルを確認
4. 速度ゾーン別の推奨負荷を確認

### フロー 4: 過去セッション確認
1. 履歴画面を開く
2. カレンダーから日付を選択
3. セッション詳細を表示
4. セット / レップ詳細を確認

## デザイン原則

### 1. 視認性
- 速度表示は巨大に（トレーニング中の視認性重視）
- 重要な情報は画面上部に配置
- コントラスト比は WCAG AA 以上を維持

### 2. 片手操作
- 重要なボタンは画面下部に配置
- タップターゲットは最小 44×44pt
- スワイプジェスチャーは直感的に

### 3. フィードバック
- ボタンタップ時のハプティックフィードバック
- セット終了時の音声フィードバック
- PR検知時のアニメーション通知

### 4. 効率性
- セッション開始から計測開始までの手数を最小化
- よく使う機能へのアクセスを優先
- データ入力は必要最小限に

## 注意事項

- **BLE接続**: iOS 13+ で Bluetooth Always 権限が必須
- **バックグラウンド**: トレーニング中のバックグラウンド実行対応
- **バッテリー**: BLE接続時のバッテリー消費を最小化
- **データプライバシー**: トレーニングデータはローカルに保存（クラウド同期は将来）
